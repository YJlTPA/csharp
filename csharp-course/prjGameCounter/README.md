# prjGameCounter — Устный Счёт

Игра для тренировки устного счёта. Два варианта интерфейса: консоль и WinForms.

## Требования

- **Windows 10/11**
- **[.NET SDK 9+](https://dotnet.microsoft.com/download)** — нужен для сборки и запуска

Проверить, установлен ли .NET:
```
dotnet --version
```
Если команда не найдена — скачать и установить SDK по ссылке выше.

## Запуск

Открой терминал (PowerShell или cmd) в папке проекта:

```
cd путь\к\csharp-course\prjGameCounter
```

### Консольная версия
```
dotnet run --project ConsoleGame
```

### WinForms версия
```
dotnet run --project WinFormsGame
```

> При первом запуске .NET автоматически скачает зависимости — нужен интернет.

## Запуск через Visual Studio

1. Открой файл `prjGameCounter.sln`
2. В **Обозревателе решений** правой кнопкой на `ConsoleGame` или `WinFormsGame`
3. Выбери **«Назначить запускаемым проектом»**
4. Нажми **F5**

## Правила игры

- Выбери **уровень сложности** (числа до 20 / 40 / 60 / 100)
- Выбери **типы операций** (+, -, *, /)
- На каждый пример отводится **15 секунд**
- За правильный ответ начисляются **монеты** с нарастающим бонусом: +1, +2, +4, +8…
- За неправильный — штраф: −1, −2, −4, −8…
- В конце показывается результат и точность

## Структура проекта

```
prjGameCounter/
├── GameCore/        — логика игры (общая библиотека)
├── ConsoleGame/     — консольный интерфейс
└── WinFormsGame/    — графический интерфейс (Windows Forms)
```
