# KbmTrainer — Тренажёр набора текста

WPF-приложение для тренировки скорости и точности печати. Вдохновлено [klavogonki.ru](https://klavogonki.ru/).

## Возможности

- **Выбор словаря** перед каждой сессией (английские слова, русские слова, цифры, программирование)
- **Редактор словарей** — добавление, переименование, удаление словарей и слов
- **Статистика** — WPM, точность, история сессий с фильтрацией
- **Настройки** — размер шрифта, тема (Dark/Light), длина текста
- Цветовая индикация: зелёный — правильно, красный — ошибка, подсветка текущего символа
- Тёмная (Catppuccin Mocha) и светлая темы

## Требования

- Windows 10/11
- [.NET 10 SDK](https://dotnet.microsoft.com/download/dotnet/10.0) (только для сборки)

## Запуск готового бинарника

Скопируй `publish/KbmTrainer.WPF.exe` в любую папку и запусти — установка .NET не требуется.

## Сборка из исходников

```bash
cd C:/dev/csharp-projects/griha/csharp/kbm-trainer

# Сборка
dotnet build KbmTrainer.sln

# Запуск
dotnet run --project KbmTrainer.WPF

# Публикация в один exe (self-contained)
dotnet publish KbmTrainer.WPF -c Release -r win-x64 --self-contained true -p:PublishSingleFile=true -o ./publish
```

## Как пользоваться

1. **Выбери словарь** на главном экране и нажми **Начать**
2. Начни печатать — таймер стартует с первым символом
3. Зелёный — верно, красный — ошибка; `Backspace` исправляет последний символ
4. По завершении текста отображается итоговый экран с WPM и точностью
5. **Словари** — добавляй/редактируй свои наборы слов
6. **Статистика** — смотри историю всех сессий
7. **Настройки** — меняй тему, размер шрифта, длину текста

## Данные

Все данные хранятся в `%APPDATA%/KbmTrainer/`:

| Файл | Содержимое |
|------|-----------|
| `dictionaries.json` | Словари и слова |
| `statistics.json` | История сессий |
| `settings.json` | Настройки приложения |
